## CherryDE

### About CherryDE
CherryDE is a BSD-focused desktop environment built for [Arisblu (AKA ArisbluBSD)](https://arisblu.com). CherryDE is built with Qt, with heavy inspiration from the [Lumina](https://lumina-desktop.org) and [Pangolin UI](https://web.dahliaos.io/#/).
